### What was involved in developing the app:

- React Hook
- React Router v6
- CSS-Modules / SCSS
- Redux-Toolkit
- mockAPI
- TypeScript
- Axios + Fetch
- React Content Loader
- React Pagination
- Lodash.Debounce
- LocalStorage
- Code Splitting, React Loadable, useWhyDidYouUpdate
